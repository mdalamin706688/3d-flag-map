# 3D Flag Map

An interactive 3D world map explorer with country flags. Built with D3.js, featuring smooth zoom transitions, continent filtering, and a premium dark-themed UI.

## Features

- Interactive world map with smooth zoom and pan
- Country hover highlights with flag tooltips
- Continent-based filtering and navigation
- Search functionality to find countries
- Detailed country info panel with flag display
- Premium dark cinematic UI with glassmorphism effects

## Tech Stack

- **D3.js** — Map rendering and interactions
- **TopoJSON** — World atlas data
- **Vanilla JS / CSS / HTML** — No frameworks

## Getting Started

```bash
# Clone the repo
git clone https://github.com/mdalamin706688/3d-flag-map.git
cd 3d-flag-map

# Serve locally
python3 -m http.server 8080

# Open http://localhost:8080 in your browser
```
